El programa calcula el ángulo de manteo aparente a partir del manteo real y alpha.

La calculadora de manteo aparente utiliza las librerías tkinter, numpy.
Dichas librerías deben ser instaladas para que el programa funcione.

Se pueden instalar facilmente de las formas:

pip install tkinter
pip install numpy

Eso básicamente.
